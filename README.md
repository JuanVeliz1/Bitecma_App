# Bitecma App

Proyecto con dos frentes principales:

- `Bitecma_Android`: aplicacion Android usada para operaciones en terreno.
- `src`: aplicacion web construida con React y Vite.

## Web

- Instalar dependencias: `npm install`
- Ejecutar en desarrollo: `npm run dev`
- Generar build: `npm run build`

## Android

- Abrir `Bitecma_Android` en Android Studio.
- Compilar desde Android Studio o con `./gradlew.bat :app:assembleDebug`

## Nota

- Este repositorio fue limpiado para entrega y ya no incluye archivos de debug o plantillas genericas no relacionadas al proyecto.
